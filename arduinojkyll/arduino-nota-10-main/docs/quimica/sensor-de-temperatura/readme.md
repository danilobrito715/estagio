# Sensor de Temperatura

Este experimento usa o Arduino para monitorar a temperatura com um sensor DS18B20 e apresentar conceitos de reações endotérmicas e exotérmicas.

## Objetivo

- Mostrar como sensores podem coletar dados térmicos.
- Relacionar medições de temperatura a reações químicas e mudanças de estado.
- Oferecer um exemplo de uso de sensores em aulas de Química.

## Arquivo de código

- `sensor_de_temperatura.ino`

## Como usar

1. Conecte o sensor DS18B20 ao pino digital 2 do Arduino.
2. Configure a leitura no Arduino IDE.
3. Observe as mensagens no monitor serial e o estado exibido no display.

## Observação

Este projeto também pode ser útil para discutir a qualidade do ar, aquecimento global e aplicações práticas de medições em laboratório.
