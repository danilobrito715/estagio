# Matemática

Esta área apresenta projetos de Arduino que ajudam a ensinar conceitos matemáticos de forma visual e prática.

## Projetos disponíveis

- [Ângulos](angulos/index.md)

## Objetivos

- Usar o Arduino para demonstrar relações entre medidas angulares e movimento.
- Aproximar a geometria e trigonometria de uma experiência prática.
- Oferecer material de apoio para professores e estudantes.
