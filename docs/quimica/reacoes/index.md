# Química — 1º Ano: Reações

Este projeto usa o Arduino para um experimento de temperatura que pode ser integrado a aulas de Química.

## Objetivo

- Demonstrar como sensores podem coletar dados de temperatura.
- Relacionar aquecimento e resfriamento a processos endotérmicos e exotérmicos.

## Arquivo de código

- [`sensor_de_temperatura.ino`](sensor_de_temperatura.ino)

## Como usar

1. Conecte o sensor DS18B20 ao pino digital 2 do Arduino.
2. Use o display LCD I2C para mostrar a leitura de temperatura.
3. Observe a diferença entre temperaturas menores e maiores que 25 ºC.
