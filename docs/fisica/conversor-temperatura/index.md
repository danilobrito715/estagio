# Conversor de Temperatura

Este projeto apresenta duas versões de conversor de temperatura usando Arduino: uma com display LCD e outra apenas com saída serial.

## Objetivos

- Demonstrar leitura de temperatura com sensor DS18B20.
- Converter valores entre Celsius, Kelvin e Fahrenheit.
- Relacionar o projeto a fenômenos físicos de calor e mudança de temperatura.

## Arquivos do projeto

- `conversor_de_temperatura_com_lcd/conversor_de_temperatura_com_lcd.ino`
- `conversor_de_temperatura_sem_lcd/conversor_de_temperatura_sem_lcd.ino`

## Como usar

1. Abra o código com o Arduino IDE.
2. Monte o circuito com o sensor DS18B20 e, se disponível, o display LCD I2C.
3. Carregue o programa no Arduino e observe as temperaturas no monitor serial ou no display.

## Observação

Este projeto pode ser usado em atividades de Física para explicar temperatura, escalas térmicas e dissipação de calor.
