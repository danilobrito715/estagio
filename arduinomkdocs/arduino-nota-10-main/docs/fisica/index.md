# Física

Esta área conecta o trabalho com Arduino ao ensino de Física.
Os projetos em Física ajudam a explicar fenômenos como movimento, forças, luz e temperatura.

## Projetos disponíveis

- [Cores](cores/index.md)
- [Força](forca/index.md)
- [Conversor de Temperatura](conversor-temperatura/index.md)
- [Lançamento Vertical](mecanica/lancamento-vertical/index.md)

## Objetivos

- Mostrar como o Arduino pode apoiar o ensino de conceitos físicos.
- Trazer exemplos práticos que dialogam com a BNCC.
- Preparar materiais para uso em sala de aula.
