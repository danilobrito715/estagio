# Requisitos

Para iniciar com Arduino em sala de aula, o projeto pede uma configuração básica:

- Arduino Uno ou placa compatível
- Cabo USB para alimentação e programação
- Protoboard (breadboard)
- LEDs, resistores 220 Ω, fios de conexão
- Sensor LDR para projetos de luz
- Sensor de temperatura DS18B20 para projetos de química
- Software: Arduino IDE ou Tinkercad Circuits

## Recomendações

- Use o Tinkercad para introduzir os conceitos sem precisar montar o circuito físico imediatamente.
- Consulte sempre o código e a lista de componentes antes de iniciar a aula.
