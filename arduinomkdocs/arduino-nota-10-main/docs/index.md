# Arduino Nota 10

Bem-vindo ao portal de documentação do projeto Arduino Nota 10.

Este site foi criado para apresentar o material didático aos professores e explicar como usar os projetos Arduino em sala de aula.

## O que você encontra aqui

- Introdução ao Arduino para professores iniciantes.
- Exemplos didáticos em `Primeiros Passos`.
- Projetos de `Física` com propostas para a BNCC.
- Um exemplo de `Química` com sensor de temperatura.
- Estrutura preparada para receber novas matérias como `Geografia`.

Use o menu à esquerda para navegar pelo conteúdo.
